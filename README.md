# React Course - Mini Projects

A collection of mini projects built during my React course. Each project focuses on different React concepts and features.

## Projects

| Project | Description |
|--------|-------------|
| 📝 `blog-application` | A simple blog app with posts and comments |
| 🐛 `bug-blaster` | A bug tracking mini application |
| 🔗 `context-api-app` | Demo of React Context API usage |
| 🎬 `moviedux` | A movie browsing app |

## Technologies Used

- **React** - UI library
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **React Context API**

## Getting Started

Each project is independent. To run any of them:

```bash
cd <project-name>
npm install
npm start
```

## Purpose

> These projects were created for **learning purposes** as part of a React course.  
> Each mini project explores different React concepts such as components, state, props, context, and more.
